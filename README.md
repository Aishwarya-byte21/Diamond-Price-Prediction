# Diamond Price Prediction

## Objective
To predict diamond prices using machine learning algorithms.

## Dataset
Contains features like carat, cut, color, clarity, depth, table.

## Algorithms Used
- Linear Regression
- Random Forest
- XGBoost

## Tools
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## Result
Random Forest gave the best accuracy.

1. Install requirements
